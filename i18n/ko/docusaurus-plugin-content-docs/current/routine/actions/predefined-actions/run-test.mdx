# dogu-actions/run-test

---

프로젝트 저장소에서 테스트 스크립트를 실행합니다.

#### Source code

[github link](https://github.com/dogu-actions/run-test)

#### Usage

```yaml
# task definition...
steps:
  - name: Run Test
    uses: dogu-actions/run-test
    with:
      environment: node
      command: |
            npm install
            npm run test:app
```


## Git Options

### checkout

<code
  style={{ position: 'relative', top: '-20px', fontSize: '90%' }}
  children="optional"
/>

테스트 스크립트를 실행하기 전에 프로젝트 저장소를 체크아웃합니다.

#### Default

false

### branch

<code
  style={{ position: 'relative', top: '-20px', fontSize: '90%' }}
  children="optional"
/>

checkout을 진행할 git branch입니다. 만약 tag가 설정되어있다면 branch는 무시됩니다.


#### Default

main


### tag

<code
  style={{ position: 'relative', top: '-20px', fontSize: '90%' }}
  children="optional"
/>

checkout을 진행할 git tag입니다. 만약 tag가 설정되어있다면 branch는 무시됩니다.


#### Default

main

### clean

<code
  style={{ position: 'relative', top: '-20px', fontSize: '90%' }}
  children="optional"
/>

체크아웃하기 전에 프로젝트 저장소를 정리합니다. checkout이 false인 경우 이 옵션은 무시됩니다.

#### Default

true

### checkoutPath

<code
  style={{ position: 'relative', top: '-20px', fontSize: '90%' }}
  children="optional"
/>

프로젝트 저장소를 체크아웃하는 경로입니다. checkout이 false인 경우 이 옵션은 무시됩니다.
이 경로는 루틴 작업공간 디렉토리로 부터의 상대경로입니다.

#### Default

''

### checkoutUrl

<code
  style={{ position: 'relative', top: '-20px', fontSize: '90%' }}
  children="optional"
/>

프로젝트 저장소를 체크아웃하기 위한 URL입니다. checkout이 false인 경우 이 옵션은 무시됩니다.
설정하지 않은 경우 Dogu 프로젝트 설정의 저장소 URL을 사용합니다. [문서](/management/organization/git-integration)를 참고하여 Dogu 조직 설정의 저장소 URL을 사용하는 것을 권장합니다.

#### Default

''
