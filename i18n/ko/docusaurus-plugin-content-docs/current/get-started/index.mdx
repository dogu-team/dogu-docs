Dogu는 웹, 앱, 게임 애플리케이션을 다양한 환경에서 테스트할 수 있도록 돕는 통합 테스트 플랫폼입니다.  
통합 테스트 플랫폼은 테스트를 위한 인프라 구축 및 관리를 단순화하고, 테스트 자체에 집중할 수 있도록 돕습니다.

## Dogu를 써야하는 이유

### 테스트 환경 구축 및 관리
다양한 모바일 기기를 구매하고, 테스트를 위한 인프라를 구축하고, 테스트 환경을 관리하는 것은 매우 복잡한 작업입니다. 또한 테스트 환경을 구축하고 관리하는 데 필요한 비용은 매우 높습니다.  
Dogu는 이러한 작업 없이 클라우드 기반의 테스트 환경을 제공하므로 테스트 환경 구축 및 관리에 소요되는 비용을 절감할 수 있습니다.

### 테스트 자동화
테스트 자동화를 위해 Selenium, Appium, OpenSTF, Jenkins, Grafana와 같은 인프라 구성 요소를 구축하고 유지 관리하려면 상당한 시간과 노력이 필요합니다.  
Dogu는 이러한 인프라 구성 요소를 제공하기 때문에 테스트 자동화에 필요한 시간과 노력을 절감할 수 있습니다.
