# 고급 설정

## 절전 모드 설정

Windows 절전 모드에 진입할 경우 의도치 않게 Dogu-Agent의 동작이 멈춤 수 있습니다. [문서](https://support.microsoft.com/en-us/windows/how-to-adjust-power-and-sleep-settings-in-windows-26f623b5-4fcc-4194-863d-b824e5ea7679)를 참고하여 절전모드에 진입하지 않도록 설정해주세요.

## 소프트웨어 업데이트

소프트웨어 업데이트가 켜져있을 경우 의도치 않게 Dogu-Agent의 동작이 멈춤 수 있습니다. Windows 업데이트를 막는 것은 좋은 방법이 아니지만, 막야아 하는 상황이라면 [Wu10Man](https://github.com/WereDev/Wu10Man) 등의 소프트웨어를 사용하여 업데이트를 막을 수 있습니다.
