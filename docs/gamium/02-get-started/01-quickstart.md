# 소개
gamium은 테스트하고자 하는 동작을 스크립트로 작성할 수 있게 해 줍니다. 테스트 스크립트는 gamium-controller를 통해 실행됩니다. gamium-controller는 gamium-agent를 통해 테스트 대상 게임에 명령을 전달합니다. gamium-agent는 테스트 대상 게임의 UI를 분석하여 gamium-controller에게 전달합니다. gamium-controller는 gamium-agent로부터 전달받은 UI 정보를 통해 테스트 스크립트를 실행합니다.
